Esta pasta contém os arquivos de som do jogo.

Arquivos necessários:
- menu.wav: Música de fundo do menu
- investigacao.wav: Música de fundo durante a investigação
- interrogatorio.wav: Música de fundo durante interrogatórios
- descoberta.wav: Efeito sonoro ao encontrar uma pista
- acusacao.wav: Efeito sonoro ao fazer uma acusação

Você pode adicionar seus próprios arquivos de som, mas mantenha os mesmos nomes para que o jogo funcione corretamente.

Formato recomendado: WAV, 44.1kHz, 16-bit, estéreo.
